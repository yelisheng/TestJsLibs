#### vscode快捷键
 + 移动光标
    + Cmd + ←	Fn + ← 将光标定位到当前行的最左侧 
    + Cmd + →	Fn + → 将光标定位到当前行的最右侧 
 + 删除操作
    + option + Backspace	Ctrl + Backspace 删除光标之前的一个单词 
    + Cmd + delete 删除光标之前的整行内容 

 + 编辑操作
    + Cmd + Enter	Ctrl + Enter	在当前行下面新增一行，然后跳至该行
    + Option + ↑	Alt + ↑	将代码向上移动
    + Option + ↓	Alt + ↓	将代码向下移动
    + Option + Shift + ↑	Alt + Shift + ↑	将代码向上复制
    + Option + Shift + ↓	Alt + Shift + ↓	将代码向下复制

 + JS语言相关
    + Cmd + /	Ctrl + /	添加单行注释
    + Option + Shift + F	Alt + shift + F	代码格式化

 + 跳转操作
    + Ctrl + Tab	Ctrl + Tab	在已经打开的文件之间进行跳转	
    + Cmd + P	Ctrl + P	在当前的项目工程里，全局搜索文件
    + Ctrl + G	Ctrl + G	跳转到指定行

 + 搜索

    + Cmd + G	F3	在当前文件中搜索，光标仍停留在编辑器里
    + Cmd + Shift + F	Ctrl + Shift +F	全局搜索

 + 多光标的使用
    + 技巧1：按住 「Option」键（windows 用户是按住「Alt」键） ，然后在页面中希望中现光标的位置点击鼠标。
    + 技巧2：选中某个文本，然后反复按住快捷键「 Cmd + D 」键（windows 用户是按住「Ctrl + D」键）， 即可将全文中与光标当前所在位置的词相同的词逐一加入选择。
    + 技巧3：选中一堆文本后，按住「Option + Shift + i」键（windows 用户是按住「Alt + Shift + I」键），既可在每一行的末尾都创建一个光标。

 + 工作区快捷键
    + cmd +/-	ctrl +/-	将工作区放大/缩小（包括代码字体、左侧导航栏）	在投影仪场景经常用到
    + Cmd + B	Ctrl + B	显示/隐藏侧边栏
    + Cmd + Option + 左右方向键	Ctrl + Pagedown/Pageup	在已经打开的文件之间进行切换
